    Interface MEPDetails

    Detailed MEP information including biography and social media.

    Extends MEP with additional biographical information, contact details, social media profiles, voting statistics, and parliamentary roles. This interface represents the complete MEP profile available through the EP API /meps/{id}/details endpoint.

    Personal Data: Contains multiple GDPR-protected fields (phone, address) requiring audit logging and data minimization practices.

    Caching: Due to personal data, cache TTL should not exceed 15 minutes per ISMS Policy DP-003 (Data Retention).

    MEPDetails

    Example

    const mepDetails: MEPDetails = {
      // MEP base fields
      id: "person/124936",
      name: "Jane Marie Andersson",
      country: "SE",
      politicalGroup: "S&D",
      committees: ["DEVE", "ENVI"],
      email: "jane.andersson@europarl.europa.eu",
      active: true,
      termStart: "2019-07-02",
      
      // Extended fields
      biography: "Member of Parliament since 2019...",
      phone: "+32 2 28 45000",
      address: "European Parliament, Rue Wiertz 60, 1047 Brussels",
      website: "https://example.com",
      twitter: "@janeandersson",
      facebook: "jane.andersson.official",
      votingStatistics: {
        totalVotes: 1250,
        votesFor: 800,
        votesAgainst: 350,
        abstentions: 100,
        attendanceRate: 92 // 92% participation
      },
      roles: ["Vice-Chair of DEVE Committee", "Member of ENVI Committee"]
    };
